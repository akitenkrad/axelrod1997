# 論文再現 (Table 7-2)

Axelrod (1997) の中心的な定量結果は Table 7-2 です — 特徴数 $f$ と特性数 $q$ のいくつかの組合せについて，10×10 グリッド・各条件10回の平均安定文化地域数を示します．

## ベンチマーク (Table 7-2)

10×10 グリッド，各条件10回の平均安定文化地域数．

| 特徴数 $f$ | 特性数 $q$ | 論文値 | 再現目標 |
|-----------|-----------|-------|---------|
| 5 | 5 | 1.0 | ±0.3 |
| 5 | 10 | 3.2 | ±0.5 |
| 5 | 15 | 20.0 | ±3.0 |
| 10 | 5 | 1.0 | ±0.3 |
| 10 | 10 | 1.0 | ±0.3 |
| 10 | 15 | 1.4 | ±0.3 |
| 15 | 5 | 1.0 | ±0.3 |
| 15 | 10 | 1.0 | ±0.3 |
| 15 | 15 | 1.2 | ±0.3 |

## 2次元の非対称的効果

この表から，2つのパラメータの非対称的な効果が観察されます：

- **$f$（特徴数）の増加 → 地域数は減少**（反直感的）．共通点が増えるほど類似度が高まり収斂しやすくなります．
- **$q$（特性数）の増加 → 地域数は増加**（直感的）．初期多様性が高まり共通点が見つかりにくくなります．

劇的なケースは $f=5, q=15$ で，多数の安定地域が残存し，単一の共有文化に潰れることなく，グローバルな文化的分極化が内生的に生じます．

## 再現方法

`sweep` サブコマンドで Table 7-2 の全9条件（$f, q \in \{5, 10, 15\}$ の 3×3 グリッド）を各10回ずつ実行します：

```bash
cargo run --release -- sweep \
    --features-min 5 --features-max 15 --features-step 5 \
    --traits-min   5 --traits-max   15 --traits-step   5 \
    --runs 10 --seed 42
```

そして結果を可視化します．可視化スクリプトは `sweep` 結果を自動判定し，該当する場合は Table 7-2 ベンチマークとの比較も描画します．

```bash
uv run python analysis/visualize.py
```

生成される図とその解釈については [可視化](visualization.ja.md) を，全フラグ表については [CLI — `sweep`](cli.ja.md#sweep-パラメータスイープ) を参照してください．
