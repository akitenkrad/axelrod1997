# ユースケース

本プロジェクトは Axelrod (1997) "The Dissemination of Culture" の文化拡散モデルを再現実装したものです．以下に，本プロジェクトでできる典型的なことと，それぞれの詳細ドキュメントへの入口を示します．

## 1. 論文のベンチマークを再現する (Table 7-2)

Table 7-2 が報告する特徴数・特性数の組合せについて，10×10 グリッドでの平均安定文化地域数を再現し，特徴的な2次元の非対称性を確認します — 特徴数が増えると地域数は **減少**（反直感的），特性数が増えると地域数は **増加**（直感的）．

- ベースケースを実行する：[CLI — `simulate`](cli.ja.md#simulate-単一パラメータセット) を参照．
- 論文値との比較と f×q の解釈を読む：[論文再現](reproduction.ja.md) を参照．

## 2. パラメータ感度分析 (sweep)

特徴数 $f$ と特性数 $q$ をグリッド上でスイープし，平衡時の安定地域数がどう応答するかを調べます．スイープにより f×q の非対称性がヒートマップとして可視化されます．

- グリッドサーチを実行する：[CLI — `sweep`](cli.ja.md#sweep-パラメータスイープ) を参照．
- ヒートマップと周辺折れ線を可視化する：[可視化 — sweep](visualization.ja.md#sweep-の可視化) を参照．

## 3. 結果を可視化して解釈する

`simulate` や `sweep` の結果を図に変換し，定性的な挙動を読み取ります — 単一文化への収斂か，複数の安定地域の残存（グローバルな文化的分極化）か．

- 最新実行から図を生成する：[可視化](visualization.ja.md) を参照．
- `metrics.csv` のカラム参照と「典型的な結果の読み方」を読む：[可視化 — 出力の解釈](visualization.ja.md#出力の解釈) を参照．
